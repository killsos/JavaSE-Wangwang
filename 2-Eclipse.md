2.3.2 Eclipse 快捷键

	alt + /         代码提示
	
	cmd + n         新建
	
	cmd + F6        在打开的文件间进行切换
	
	cmd + F7        在资源窗口间进行切换
	
	cmd + F8        在各种模式下间进行切换
	
	cmd + e         选择某个打开的文件
	
	cmd + shift + t 打开一个类   // open type
	
	cmd + shift + r 打开一个文件 
	
	cmd + shift + f 格式化代码
	
	cmd + d         删除当前行
	
	cmd + 1         代码错误解决方法提示
	
	cmd + o         查看类结构
	
	cmd + shift + o         导入需要的类
	
	cmd + t          列出子类或接口实现类(类的等级结构)
